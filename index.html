Problème de réseau informatique

Bonjour sur l'application de l'implémentation du problème de plus cours chemain.
Cette application vous permet la simulation d'un cas problème de plus cours chemin(par exemple le plus cours chemin dans un réseau informatique pour que les données envoyées arrivent dans le plus tôt possible).

Pour commancer une nouvelle simulation, clicker sur OK.
